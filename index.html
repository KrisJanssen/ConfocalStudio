Welcome to Confocal Studio!

Confocal Studio is an open source project, designed to enable recording confocal images with any type of hardware in custom setups.

Confocal Studio was developed in C# .NET 4.5 using VS 2013.

Here we leverage the wonderful Gemini UI framework developed by @tgjones, allowing for a modular design.

Gemini in turn is developed using the Caliburn.Micro MVVM framework.
